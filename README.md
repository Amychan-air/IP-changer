# IP Changer - Linux 网络配置管理面板

这是一个基于 Node.js 的 Web 面板，可以通过 SSH 远程管理 Linux 服务器（支持 Ubuntu, Debian, CentOS 等）的 IP 地址和网络配置。

## 功能特点

- **多系统支持**：自动识别并适配 Ubuntu (Netplan), Debian (Interfaces), CentOS (NetworkManager/nmcli) 等。
- **IP 管理**：可视化添加、删除静态 IP 地址。
- **网关与 DNS**：轻松配置默认网关和 DNS 服务器。
- **批量操作**：支持 CIDR 范围解析并批量应用。
- **内置终端**：集成了基于 Web 的 SSH 终端，方便直接执行命令。
- **连接测试**：支持指定源 IP 进行 Ping 测试，确保配置生效。

## 安装步骤

### 前置要求
- 已安装 [Node.js](https://nodejs.org/) (建议 v16 或更高版本)。
- 目标 Linux 服务器需开启 SSH 服务。

### 1. 克隆/下载项目
```bash
git clone <项目地址>
cd IP-changer
```

### 2. 安装依赖
```bash
npm install
```

### 3. 启动应用
```bash
npm start
```
默认情况下，应用将在 `http://localhost:3000` 运行。

## 使用说明

### 1. 连接服务器
- 打开浏览器访问 `http://localhost:3000`。
- 在页面左侧点击 **"Add Host"**。
- 输入服务器的 **IP 地址**、**SSH 端口**（默认 22）、**用户名** 以及 **密码** 或 **私钥**。
- 点击 **"Connect"**。系统会自动检测服务器的发行版并加载当前的网络接口信息。

### 2. 查看网络状态
- 连接成功后，中间面板会显示所有网卡（如 `eth0`, `ens33`）及其当前的 IP 地址、子网掩码和网关。

### 3. 修改 IP 配置
- **添加 IP**：在 "Apply Static Config" 区域输入 IP 范围（如 `192.168.1.10/24`）或单个 IP，选择网卡并点击 **"Apply"**。
- **删除 IP**：在接口详情中，点击对应 IP 旁的删除图标。
- **更新 DNS/网关**：修改对应输入框后点击提交。

### 4. 使用终端
- 点击右上角的 **"Terminal"** 按钮，可以打开一个与服务器同步的命令行窗口，进行更高级的操作。

### 5. 测试连接
- 在 IP 列表旁点击 **"Ping"** 图标，系统会尝试使用该 IP 作为源地址去 ping 8.8.8.8，并显示延迟。
